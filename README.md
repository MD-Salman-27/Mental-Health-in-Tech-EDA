## Project Overview
This project performs Exploratory Data Analysis (EDA) on the Mental Health in Tech Survey (2014) dataset to analyze employee awareness, treatment behavior, and workplace support for mental health.

## Dataset
- Source: Open Sourcing Mental Illness (OSMI)
- Year: 2014
- Records: 1259

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## Key Analysis
- Univariate Analysis
- Bivariate Analysis
- Multivariate Analysis

## Conclusion
The analysis highlights the importance of employer support and mental health benefits in improving employee well-being in the tech industry.
